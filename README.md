# udacity-gen-ai

## Create virtual environment

```
$ python -m venv .venv
```

## Create or Update requirements.txt

```
$ pip freeze > requirements.txt
```

## Install

```
$ pip install -r requirements.txt
```

## Start Jupyter Notebook

```
$ jupyter-notebook
```

## Activate/Deactivate venv

```
// activate venv
$ source .venv/bin/activate

// deactivate venv
$ deactivate
```
